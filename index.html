About Me

In my free time, I love exploring new frameworks and best practices to keep up with the latest trends in web development. I’m excited to work on projects that challenge me and contribute to innovative web experiences.

My Project

Project Description:- Zomato Clone This project is a basic structural clone of the Zomato website, created using only HTML and CSS. It focuses on building the core layout, including headers, sections, and footers, to mimic the appearance of Zomato's interface. The goal is to create a visually similar design with a clean and organized structure, showcasing my skills in HTML and CSS.

Project Description:- BookMyShow Clone This project involves creating a structural replica of the BookMyShow website using only HTML and CSS. The focus is on building the layout for movie categories, show timings, and user interaction sections like booking and filters. The project emphasizes the organization of content and responsive design, ensuring the layout is clean and adapts well to different screen sizes. This clone showcases my proficiency in structuring web pages and creating user-friendly interfaces with just HTML and CSS.
